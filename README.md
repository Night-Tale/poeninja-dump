Итак, что тебе нужно сделать:
- На своем диске создай таблицу для работы
- Открой таблицу, в ней тебе нужно создать лист, на который будет происходить выгрузка и запомнить его название, а также скопировать id таблицы из ссылки(https://docs.google.com/spreadsheets/d/ВОТ_ЗДЕСЬ_ID/edit?gid=0#gid=0)
- Склонируй себе репо
- В папке config внеси id таблицы и название листа для выгрузки в кавычки в файл sheet_config.yml (открой любым редактором)
- Скачай файлик, который дам в телеге, и положи в корень проекта
- В таблице выдай права доступа и редактирования сервис-аккаунту (его email найди в поле client_email в этом файлике)
- В папке scripts лежат 2 файла, сначала запускаешь install (один раз, чтобы установить зависимости и виртуальное окружение, чтобы потом он тебя не смущал - можешь удалить)
- Если install отработал - можешь запускать run, он будет чистить лист, который ты укажешь в конфигах и записывать туда свежие данные, когда понадобятся свежие данные - перезапускаешь run
